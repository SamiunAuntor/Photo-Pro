# Photo-Pro

Live: https://photo-pro-wine.vercel.app/

Photo Pro is a small frontend-only passport and ID photo layout tool. The app is focused on a simple workflow: upload a background-removed photo, crop it to a fixed photo ratio, preview it, and generate a printable layout for paper sizes like A4.

## What It Does

- Upload a prepared photo
- Crop and position it inside a fixed passport-style frame
- Preview the cropped result
- Generate a repeated print layout
- Download the layout as a PDF
- Open a print-friendly sheet

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Zustand
- jsPDF

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
